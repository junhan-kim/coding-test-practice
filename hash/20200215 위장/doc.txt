[문제] https://programmers.co.kr/learn/courses/30/lessons/42578
[유형] map
[풀이]
 key : 의상 카테고리, value : 갯수
 조합이므로 map의 각 value를 곱해주면 된다. 
 단 유의할점은 두가지이다.  
 1. 안 입는다는 경우도 있으므로 초기값은 2로 해준다. (맨 처음 맵에 어떤 의상을 등록하면 그 의상을 입는 경우, 입지 않는 경우로 두가지임)
    그 이후에는 1씩 늘린다. (의상을 두개 등록하면 의상을 입지 않는 경우, A 의상을 입는 경우, B 의상을 입는 경우로 세가지임)
 2. 의상을 적어도 한개는 입어야 한다.  즉, 의상을 모두 입지 않는 경우는 없으므로 전체 결과에서 1을 빼주어야 한다.

[주요 기능]
m.find(key) == m.end()  -> 맵에 특정 키가 존재하지 않음.   // 매우 유용

[에러]
1. if else에서 else 안넣어서 초기화, 이후 대입을 구분하지 못한 실수;
