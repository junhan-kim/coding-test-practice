[문제] https://programmers.co.kr/learn/courses/30/lessons/42577
[유형] map
[풀이]
 전체 배열을 돌면서 각 문자열을 맵에 등록한다.
 다시 배열을 돌면서 각 문자열마다 앞에서부터 한글자씩 늘려가면서 맵 안에 존재하는지 확인한다.
 (만약 이를 반복문으로 처리하면 한글자씩 늘릴때마다 전체 배열을 다 돌아야함)

[주요 기능]
  str.substr(인덱스, 길이)

[에러]
1. 문제가 약간 애매모호했음. 전화번호가 중복될 수 있는지의 여부가 안 나와 있었다.
   어쨌든 문맥상으로 봤을때 중복되지 않을 것이라고 판단하고 자기 자신을 포함하는 substr을 제외하니 성공하였다.
    ex) 119 -> { 1, 11 }    // 119는 제외
